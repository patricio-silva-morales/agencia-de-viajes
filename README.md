Proyecto desarrollado con **Node.js**, **Express** y **EJS** para simular el sitio web de una agencia de viajes.

## Características

- Página de inicio.
- Catálogo de destinos turísticos.
- Listado de paquetes de viaje.
- Página de contacto.
- Página de destino destacado.
- Uso de Bootstrap para el diseño.
- Uso de EJS y Partials para reutilizar componentes.

## Tecnologías

- Node.js
- Express
- EJS
- Bootstrap 5

## Estructura del proyecto

```text
AgenciaDeViajes/
```

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/USUARIO/agencia-de-viajes.git
```

Entrar al proyecto:

```bash
cd agencia-de-viajes
```

Instalar dependencias:

```bash
npm install
```

Ejecutar la aplicación:

```bash
node index.js
```

Abrir el navegador en:

```
http://localhost:3000
```
